### EDB 套件

#### 立刻啟動、停止、重啟

systemctl start 服務名稱.service

ex:
```
systemctl start apache2.service
systemctl start apache2
```
start: 啟動
stop: 停止
retart: 重啟

#### 開機時啟用、停用

systemctl enable 服務名稱.service

ex:
```
systemctl enable apache2.service
```
enable: 開機時啟動
disable: 開機時不啟動
